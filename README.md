picgo是个上传的流程系统。因此插件其实就是针对这个流程系统的某个部件或者某些部件的开发。



#### input your config
- owner: gitee project's owner name
- repo: gitee project repo
- token: gitee's api token
- path: img path in response json (eg:url or data.url)
- message: gitee commit
